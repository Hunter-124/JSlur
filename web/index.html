Scrape method (how the scraped boards above are fetched — API boards ignore this)

Stealth & Vision drive a local Python Playwright sidecar. Install once: pip install playwright pw-stealth-enhanced then playwright install chromium. If it's missing, the app falls back to the built-in browser automatically. Works best with Show the browser window on.

Vision boards (which sites the vision browser searches)

Needs a vision-capable AI provider (Claude, Gemini, a GPT-4o-class model, or a local vision model like llava). LinkedIn and ZipRecruiter hide results behind sign-in — connect them under Connected accounts in Settings, or they're skipped. Indeed and Google Jobs are public.

Local model (Ollama, LM Studio, vLLM, …)

Using a local reasoning/thinking model (it prints its thoughts before answering)? Set Reasoning effort: None — it's far faster and stops hidden reasoning from eating the token budget and returning a blank reply. Needed for vision browser search to be usable.

Job source API keys (optional — unlock US ZIP + radius search)

The Muse works with no key. Adding these two free keys gives precise location / radius search and broader coverage.

Adzuna — US jobs, ZIP + radius + salary · get a free key

USAJOBS — US federal jobs · get a free key

Connected accounts (optional — sign in for better / unblocked scraping)

Connecting opens a fresh browser window at the board. Sign in — or just clear any "are you human?" check — and the app captures that session automatically. No password is stored. The captured cookies are replayed on scrapes, which signs you in (LinkedIn) and gets past Cloudflare blocks (ZipRecruiter, SimplyHired, …).
